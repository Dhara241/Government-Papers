# Government-Papers

A web application to view and download Gujarat Government exam papers including GPSC, Police, Clerk, and more.  
Built using **Python (Django), HTML, CSS, and JavaScript**.

---

## 🚀 How to Run the Project

This project can be executed using two methods:

---

### 🔹 Method 1: Download ZIP

1. Download the project as ZIP from GitHub  
2. Extract the folder  
3. Open terminal in project folder  
4. Create virtual environment:  
   python -m venv venv  
5. Activate environment:  
   venv\Scripts\activate  
6. Install dependencies:  
   pip install -r requirements.txt  
7. Apply migrations:  
   python manage.py migrate  
8. Run server:  
   python manage.py runserver  

---

### 🔹 Method 2: Git Clone

1. Clone the repository:  
   git clone https://github.com/Dhara241/Government-Papers.git  
2. Open project folder:  
   cd Government-Papers  
3. Create virtual environment:  
   python -m venv venv  
4. Activate environment:  
   venv\Scripts\activate  
5. Install dependencies:  
   pip install -r requirements.txt  
6. Apply migrations:  
   python manage.py migrate  
7. Run server:  
   python manage.py runserver  

---
